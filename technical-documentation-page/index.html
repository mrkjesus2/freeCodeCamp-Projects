introduction

Microsoft Visual Studio code is an extremely powerful text editor along the lines of the classic Sublime Text. It has amazing built in support for web developers.

In addtion to the built-in tools there are a vast number of extensions that can be added to extend it's functionality

The following is information that I will want to remember in case I end up having to start over from scratch. If it helps anybody else, that's wonderful.

installation

There are two different ways to install VS Code. The traditional local installation to your computer or a portable installation which allows you to use it on any computer.

* Note: these are instructions for Windows

Local Install

  1. Download the installer.
  2. Run the file that you just downloaded
  3. Follow the instructions on your screen.
  4. Congratulations, you just installed a great editor.

Portable Install

  1. Download the zipped install files from here
  2. Unzip to a folder on your portable media.
  3. In the folder where you unzipped the download create a folder named 'data'. This folder will contain all preferences, extensions, etc. This folder can be moved to other local or portable installations to easily move settings.
  4. Windows portable installations don't update automatically In order to update, repeat the above steps with the new version of VS Code and copy/move the 'data'
  5. Optional - get your settings from your local install to your portable install
    1. Copy the folder %APPDATA%\Roaming\Code to the folder that was created in the steps above and rename from "Code" to "user-data"
    2. Copy the folder %USER%\.vscode\extensions to the same 'data' folder making sure to retain the folder name of 'extensions'.
    3. * note - Temporary files will still be stored on the computer hosting the portable installation. If this is undesirable create a 'tmp' directory inside the 'data' folder.

built in features

VS Code has a number of built in time saving features for the web developer. Some of the key features are Intellisense, code refactoring, emmet, snippets, etc.

Intellisense

Intellisense is intelligent code completion and is built into VS Code for Javascript. Other languages can be enabled with extensions. The beautiful thing about Intellisense is that it will also suggest and auto-complete functions defined in your project.

For javascript the only thing you need to know is that VS Code will pop up suggestions as you type and to accept the suggestion you only need to press the "tab" key.

shortcuts

There are a number of extremely useful shortcuts that help to speed things up for the developer. If you use the shortcuts ctrl + k ctrl + r your browser will open a window with a cheat sheet.

Here is a list of the shortcuts I find most useful:

  • ctrl + d - when you select text, this will add the next occurrence to your selection
  • alt + up/down - Move the current line up or down
  • ctrl + ` - Open a terminal in the current folder
  • ctrl + Enter - Start a new line below
  • ctrl + Shift + Enter - Start a new line above
  • ctrl + / - Toggles comment cursor's line
  • ctrl + Shift + p - Open the command palette

extensions

Extensions are a great feature that allows the developer to save themselves time if they find the right tool.

To install an extension:

  1. Click the extensions icon in the activity bar
  2. search for an extension that you would like and click on it
  3. Read the documentation for any specific instructions that may be required
  4. Click the install button

Webhint

Checks code as you write and issues errors and warnings when it finds problems in your code. There are an incredible number of rules that can be enabled/disabled by the user.

It is my recommendation that you enable the nu html checker for W3C validation. This is done by adding `html-checker` to the hints key in your ".hintrc" file. For further information on rules that can be added check the webhint website.

Live Server

Runs a server on port 5500 that will live reload the page when you save your file. Simply press the "Go Live" option in the status bar of VS Code.

Browser Preview

Used in conjuction with live server allows you to see live changes in VS Code itself. Just press the "Browser Preview" button in the Activity bar an put localhost:5500/your-direcory-here in the browser that appears.
