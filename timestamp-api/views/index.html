API Project: Timestamp Microservice

A project for freeCodeCamp. It is an API that takes a date string or unix time and returns a json response with the unix time and utc date string.

Example Usage:

  • [project url]/api/timestamp/2015-12-25
  • [project url]/api/timestamp/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
